# Gold Testset 기여 가이드

Garu 골드 테스트셋은 5,000문장의 한국어 형태소 분석 데이터셋입니다.
오류 수정과 문장 추가 두 가지 방식으로 기여할 수 있습니다.

---

## 오류 수정

`gold_testset.jsonl`에서 잘못된 형태소 분석을 발견하면 GitHub Issue로 제보해주세요.

**Issue 제목:** `[오류] <문장 원문>`

**Issue 본문:**
```
문장: 배가 아파서 약을 먹었다
현재 분석: [["배", "NNG"], ["가", "JKS"], ...]
올바른 분석: [["배", "NNG"], ["가", "JKS"], ...]
근거: (간단한 설명)
```

- 형태소 표기는 `["형태소", "POS"]` 형식으로 작성
- 근거는 한 줄로도 충분합니다

---

## 문장 추가

새 문장을 추가하려면 PR을 제출해주세요.

### 형식

`gold_testset.jsonl`에 다음 형식으로 추가합니다:

```jsonl
{"text": "문장", "morphemes": [["형태소", "POS"], ...], "domain": "도메인", "confidence": "manual"}
```

**domain 값:** `뉴스` / `일상` / `SNS` / `기술` / `문학` / `엣지케이스`

엣지케이스 문장은 `edge_category` 필드도 포함해주세요:
```jsonl
{"text": "나는 하늘을 나는 새를 봤다", "morphemes": [...], "domain": "엣지케이스", "edge_category": "동형이의어", "confidence": "manual"}
```

**edge_category 값:** `동형이의어` / `단음절다의어` / `VCP_JKS` / `XSV_VV` / `XSA_VA` / `ETM_JX` / `접사_단독` / `종결_연결` / `외국어_혼합` / `OOV_고유명사`

### 조건

- 문장 길이: 5~50자
- 실제 한국어에서 자연스럽게 나올 법한 문장
- 중복 문장 없음 (`text` 기준)

## 세종 태그셋 요약

| 분류 | 태그 |
|------|------|
| 체언 | NNG(일반명사) NNP(고유명사) NNB(의존명사) NR(수사) NP(대명사) |
| 용언 | VV(동사) VA(형용사) VX(보조용언) VCP(이다) VCN(아니다) |
| 수식언 | MAG(부사) MAJ(접속부사) MM(관형사) IC(감탄사) |
| 조사 | JKS JKC JKG JKO JKB JKV JKQ(격조사) JX(보조사) JC(접속조사) |
| 어미 | EP(선어말) EF(종결) EC(연결) ETN(명사형) ETM(관형형) |
| 접사 | XPN(접두) XSN XSV XSA(파생접미) XR(어근) |
| 기호 | SF SP SS SE SO SW SH SL SN |

### 자주 헷갈리는 패턴

| 패턴 | 분석 |
|------|------|
| `학생이다` | 학생/NNG + 이/VCP + 다/EF |
| `학생이 간다` | 학생/NNG + 이/JKS + 가/VV + ㄴ다/EF |
| `공부하다` | 공부/NNG + 하/XSV + 다/EF |
| `일을 하다` | 일/NNG + 을/JKO + 하/VV + 다/EF |
| `필요하다` | 필요/NNG + 하/XSA + 다/EF |
| `먹는 밥` | 먹/VV + 는/ETM + 밥/NNG |
| `나는 학생` | 나/NP + 는/JX + 학생/NNG |
| `저공해` | 저/XPN + 공해/NNG |
| `저 사람` | 저/MM + 사람/NNG |
